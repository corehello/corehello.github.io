On my way to Common Lisp

Common Lisp is a programming language, and the first functional programming language. It has a long history and can back track to early of last century. It come from Mathmatics(which is called lambda calculus invented by Alonzo Church). It is the second-oldest high-level language and can support nearly all paradigms today(actually we can say all paradigm can implemented by common Lisp). And has all complex data structure which other programming language supported in mordern life.

Common Lisp has powerful macros, so it can create a language layer over the common lisp. And language is a set of expressions, so it is called expression-oriented language and even language-oriented programming language. Common Lisp is widwspread used in AI(Artificial intelligence) research. And many concepts of other programming language come from common lisp, and Common lisp have many features which influenced many other languages.

  • automatic garbage collection
  • functional programming (lambda, closure, high-order function, lazy evaluation)
  • macro
  • restart(exception handling)
  • generic setter
  • DSL(domain specific language)
  • CLOS(common lisp object system, communication with generic function)
  • continuation(time machine)
  • adapt multicore programming

History

Lisp was invented in 1958 by John McCarthy with the famous paper "Recursive Functions of Symbolic Expressions and Their Computation by Machine, Part I". But the theory have already invented by Alozon Church who try to solve the computability theory for Hilbert's 10th problems. He invented the lambda calculus system on mathmatics, and Turing inventd Turing machine on computer hardware. So as we know, knowledges which based on mathmatics will never be out of date.
