How to setup common lisp development

Step 1: Download and install Emacs

You can download and install via:

  • yum install emacs (RedHat serials, like centos, fedora)
  • apt-get install emacs (debian serials, like ubuntu, kali ...)
  • For Mac osx download emacs from here
  • For windows users(Better to switch to unix-like system, linux for mac osx)

Step 2: Install slime for emacs

Add the following line to you ~/.emacs:

(setq package-archives '(("gnu" . "http://elpa.gnu.org/packages/")
                    ("marmalade" . "https://marmalade-repo.org/packages/")
                    ("melpa" . "http://melpa.milkbox.net/packages/")))
(add-to-list 'package-archives '("marmalade" . "https://marmalade-repo.org/packages/"))
(add-to-list 'package-archives '("melpa" . "http://melpa.milkbox.net/packages/"))
(add-to-list 'package-archives '("org" . "http://orgmode.org/elpa/") t) ; Org-mode's repository

Install slime from meppa source

  • launch emacs
  • type "Alt-x", and input "package-install", then press "enter"
  • type "slime", press "enter"
  • wait some seconds(depend on your network speed)

Step 3: Install a common lisp interpreter and compiler

You can choose any software to do this, like sbcl, clisp ...
And after you install it, append the following line to ~/.emacs

(setq inferior-lisp-program "/usr/local/bin/sbcl")
(setq slime-contribs '(slime-fancy))

Step 4: Restart emacs, and start slime

  • Restart emacs

  • type "Alt-x", then "slime"
    And after all is done, there should be a prompt message come out:

    ; SLIME 2015-02-19
    CL-USER>
