How to use bash history expansion

Examples:

  1. re-execute the last commands:
    echo !! shortern for echo !-1
  2. re-execute the second last command: echo !-2
  3. re-execute the last command with string replace
    service sshd start
    ^start^stop^ equal !!:s/start/stop
  4. execute command with the last argument of the last command
    echo !$ shortern for echo !!:$
  5. execute command with the first argument of the last command
    echo !^ shortern for echo !!:1
  6. execute command with all argument of the last command
    echo !!:* shortern for !!:1-$
  7. The most complex command:
    echo !-3:2-3:gs/start/stop/
